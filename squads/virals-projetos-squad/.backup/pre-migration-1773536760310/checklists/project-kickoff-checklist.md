# Checklist: Kick-off de Projeto

**Projeto:** ___ | **PM:** ___

## Antes do Kick-off
- [ ] Project brief aprovado por todas as partes
- [ ] Buracos identificados e resolvidos
- [ ] Agenda de kick-off enviada com 24h de antecedência
- [ ] Acesso às ferramentas (ClickUp, etc.) enviado ao cliente

## Durante o Kick-off
- [ ] Objetivo e critério de sucesso alinhado
- [ ] Papéis e responsabilidades definidos
- [ ] Plano de comunicação acordado (canal, frequência, formato)
- [ ] Primeiro sprint/semana acordado
- [ ] Próxima reunião agendada

## Após o Kick-off
- [ ] Ata enviada para todos (< 24h)
- [ ] Tasks de todos os squads criadas no ClickUp
- [ ] Card no Painel de Projetos atualizado
- [ ] Marcos criados em Gestão de Projetos
- [ ] Health status definido como 🟢

**PROJETO INICIADO FORMALMENTE: [ ] Sim**
