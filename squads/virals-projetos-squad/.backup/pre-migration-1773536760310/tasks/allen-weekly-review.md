# Task: Weekly Portfolio Review

**Agent:** @allen-proj
**Trigger:** Fim de ciclo semanal (ex: Sexta-feira 16h ou Segunda-feira 8h).

## Entradas
- O panorama atual do ClickUp de Gestão de Projetos e Projetos Externos.

## Objetivo
1. Esvaziar a mente do PM/Equipe para o fim de semana.
2. Garantir que o sistema (ClickUp) está confiável (verdadeiro e atualizado).
3. Garantir que absolutamente TODO projeto ativo tem uma "Próxima Ação" clara definida.

## O que o agente deve fazer
- Passe projeto a projeto. Se houver projeto estagnado (sem atividade > 5d), sinalize o líder responsável.
- Aplique o `portfolio-health-checklist.md`.
- Constate: Todas as 'Próximas Ações' do projeto estão factíveis e agendadas?
- Há bloqueios sistêmicos externos (tarefas na lista "Aguardando Resposta")? Sinalize para cobrança.
- Promova ou rebaixe "Someday/Maybe" conforme as diretrizes passadas por Tops/Betting Table.

## Saídas
- Portfólio atualizado e sistema 'limpo'.
- Sumário de riscos e Projetos estagnados para @sutherland-proj e PM.
