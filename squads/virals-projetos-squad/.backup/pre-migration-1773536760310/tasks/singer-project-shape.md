# Task: Project Shaping

**Agent:** @singer-proj
**Trigger:** Um briefing intake preenchido com respostas às perguntas abertas.

## Entradas
- `project-brief-template.md` pré-preenchido.

## Objetivo
1. Aplicar a metodologia Shape Up: não estimar horas; definir 'Appetite' (quanto tempo queremos gastar nisto?).
2. Definir fronteiras rígidas (O que está DENTRO e o que está FORA).
3. Identificar os "Rabbit Holes" (buracos negros de tempo).

## O que o agente deve fazer
- Questionar o escopo: "Isso realmente precisa ser feito para resolver o problema núcleo?"
- Estabelecer a exclusão expressa de itens desnecessários.
- Fixar o Appetite (ex: "Big Batch de 6 semanas" ou "Small Batch de 2 semanas").
- Gerar alternativas mais simples para exigências complexas ("Podemos fazer isso com 20% do esforço gerando 80% do valor?").

## Saídas
- O `project-brief-template.md` completo e refinado.
- Preenchimento do `checklists/briefing-quality-checklist.md` validando que o brief é 'pitchable'.
