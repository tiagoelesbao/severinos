# Checklist: Sprint Planning & Readiness

**Sprint/Ciclo:** ___ | **PM/Scrum Master:** ___

## 1. Backlog Readiness
- [ ] O objetivo do sprint ("Sprint Goal") está definido e claro?
- [ ] Todos os itens selecionados ajudam a atingir o objetivo?
- [ ] Todos os itens têm um Definition of Done claro?
- [ ] A carga de trabalho é realista para o tempo disponível?

## 2. Alinhamento de Executores
- [ ] Os executores (agentes ou humanos) entenderam os itens?
- [ ] Existem pendências, bloqueios ou assets faltando para iniciar?
- [ ] A cadência de check-in (Daily/Weekly) está na agenda de todos?

## 3. Gestão ClickUp
- [ ] Tasks estão atribuídas aos donos corretos?
- [ ] Datas de início e prazo configuradas corretamente?
- [ ] Tags/Labels de sprint aplicados?
