# torres-opportunity-tree
## Responsável: @torres-produto

Construir a Opportunity Solution Tree (OST) baseada em um outcome desejado.

### Entrada
- outcome_desejado
- insights_entrevistas

### Processo
1. Definir a raiz (Outcome).
2. Mapear Oportunidades (problemas/desejos/dores).
3. Gerar Hipóteses de Solução.
4. Desenhar Experimentos para validar as soluções.

### Saída
- opportunity_solution_tree_map
