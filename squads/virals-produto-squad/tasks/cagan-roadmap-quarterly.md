# cagan-roadmap-quarterly
## Responsável: @cagan-produto

Task para criar o roadmap de produto trimestral orientado a outcomes.

### Entrada
- okrs_empresa
- feedback_vendas
- insights_discovery

### Processo
1. Revisar os OKRs da Virals para o trimestre.
2. Analisar as oportunidades priorizadas na Opportunity Solution Tree (@torres).
3. Definir os outcomes principais para Produto.
4. Distribuir iniciativas nos horizontes 1, 2 e 3.

### Saída
- roadmap_produto_trimestral
- priorizacao_backlog
