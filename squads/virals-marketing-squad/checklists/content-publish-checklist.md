# Checklist: Publicação de Conteúdo @mrbeast @garyvee

- [ ] Thumbnail gera curiosidade imediata e contraste visual?
- [ ] Título tem menos de 50 caracteres e gera CTR alto?
- [ ] Hook nos primeiros 3-5 segundos para o scroll?
- [ ] A promessa do título é entregue no início do vídeo?
- [ ] Existem micro-recompensas a cada 30-60 segundos?
- [ ] O conteúdo é nativo para a plataforma de destino?
- [ ] Existe um gatilho de compartilhamento ou CTA de engajamento?
- [ ] Legendas e tags otimizadas para busca/algoritmo?
- [ ] Qualidade técnica (áudio/vídeo) atende ao padrão Virals?
