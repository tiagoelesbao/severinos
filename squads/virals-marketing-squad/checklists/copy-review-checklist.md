# Checklist: Revisão de Copy @ladeira

- [ ] Headline atrai a atenção do público-alvo?
- [ ] O nível de consciência está corretamente endereçado?
- [ ] A Big Idea é clara e única?
- [ ] Existe uma dor/desejo central sendo explorada?
- [ ] O storytelling flui naturalmente para a oferta?
- [ ] As principais objeções foram antecipadas e quebradas?
- [ ] O CTA é único, claro e direto?
- [ ] A copy soa como uma conversa humana (psychology BR)?
- [ ] Prova social ou autoridade foi incluída?
- [ ] Existe urgência ou escassez real?
