# Checklist: Lançamento de Campanha Brand/Funil

**Campanha:** ___________________
**Objetivo:** ___________________
**Gestor:** @ezra-firestone

## Estratégia
- [ ] Fase do funil definida (topo/meio/fundo)
- [ ] Mensagem alinhada com a temperatura da audiência
- [ ] KPIs corretos definidos (não usar CPA para awareness)
- [ ] Budget e prazo aprovados

## Audiências
- [ ] Audiências personalizadas configuradas (website, video views, engajamento)
- [ ] Lookalikes criados a partir de audiências validadas
- [ ] Exclusões aplicadas (compradores existentes excluídos de topo de funil)
- [ ] Frequência máxima configurada

## Criativos
- [ ] Conteúdo nativo para a plataforma (não copiar de outra rede)
- [ ] Formato correto (vertical para feed/stories, horizontal para YouTube)
- [ ] Hook < 3s forte (vídeos)
- [ ] Proporção de valor vs. CTA correta para a fase do funil

## Rastreamento
- [ ] Pixel de evento configurado corretamente por etapa do funil
- [ ] UTMs com identificação de fase (ex: utm_content=tofu-video1)
- [ ] Benchmark de CPM/ThruPlay documentado para comparação futura
