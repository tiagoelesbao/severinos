# Architecture - Virals Sales Squad

## Design Philosophy
A "Máquina de Vendas" (Sales Machine) is designed for predictability, scalability, and cultural resonance. It separates the cognitive load of prospecção (Aaron Ross), follow-up (Jeb Blount), and closing (Jordan Belfort), while adapting everything to the Brazilian WhatsApp-first reality (Thiago Reis).

## Squad Tiers
- **Tier 0 (Orchestration):** `@sales-chief` - Routes leads and monitors pipeline velocity.
- **Tier 1 (Core Experts):** 
  - `@hormozi-sales` (Offer/Pricing)
  - `@ross-sales` (Pipeline Architecture)
  - `@blount-sales` (Prospecting/Follow-up)
  - `@belfort-sales` (Closing/Scripts)
  - `@thiago-reis` (WhatsApp/BR Context)

## Workflow Engines
1. **Predictable Revenue Engine:** Specialization of SDR and Closer roles.
2. **Straight Line Closing:** Linear persuasion path for high ticket.
3. **Fanatical Follow-up:** High-frequency, multichannel touchpoint system.

## Data Flow
`Marketing Leads -> [@sales-chief] -> [SDR (@ross/@blount)] -> [Closer (@belfort/@thiago)] -> [Closed Deal]`

## Integration Standards
- **Tasks:** HO-TP-001 (8-field) compliance.
- **Agents:** AIOX 6-level loader structure.
- **CRM:** Deep integration with ClickUp for pipeline hygiene.
