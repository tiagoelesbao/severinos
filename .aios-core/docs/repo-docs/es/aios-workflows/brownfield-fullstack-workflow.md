## Vision General

Workflow para evolucionar aplicaciones full-stack existentes, incluyendo:

- Nuevas features
- Refactoring
- Migraciones
- Mejoras de arquitectura

## Cuando Usar

- Cambios en proyecto full-stack existente
- Features que afectan frontend + backend
- Despues de brownfield-discovery
