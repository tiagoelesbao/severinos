## Vision General

Workflow para crear backends/APIs desde cero, incluyendo:

- Setup de API/microservicio
- Diseno de endpoints
- Integracion con base de datos
- Documentacion de API

## Cuando Usar

- Nuevo backend/API
- Microservicio independiente
- Sin frontend incluido

## Agentes Participantes

| Agente | Rol |

| @architect | Diseno de servicio |
| @data-engineer | Esquema de datos |
| @dev | Implementacion |
| @qa | Tests de API |
| @devops | Deploy |
