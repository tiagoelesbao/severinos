## Vision General

Workflow para evolucionar backends/APIs existentes, incluyendo:

- Nuevos endpoints
- Refactoring de servicios
- Migraciones de datos
- Mejoras de performance

## Cuando Usar

- Cambios en backend existente
- Nuevos endpoints en API existente
- Despues de brownfield-discovery
