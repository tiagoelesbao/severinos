## Vision General

Workflow para crear aplicaciones full-stack desde cero, incluyendo:

- Setup inicial del proyecto
- Arquitectura frontend + backend
- Configuracion de base de datos
- CI/CD pipeline

## Cuando Usar

- Nuevo proyecto full-stack
- Necesita frontend + backend + database
- Sin codigo existente
