## Vision General

**@architect** (Aria) es el agente de arquitectura y diseno tecnico del AIOS, responsable por:

- Diseno de arquitectura de sistemas
- Decisiones tecnicas (ADRs)
- Patrones y estandares
- Revision de arquitectura

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*design` | Disenar arquitectura |
| `*adr` | Crear decision record |
| `*review` | Revisar arquitectura |
| `*patterns` | Sugerir patrones |
