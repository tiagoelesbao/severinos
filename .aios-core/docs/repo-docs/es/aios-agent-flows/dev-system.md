## Vision General

**@dev** (Dex) es el agente de desarrollo del AIOS, responsable por:

- Implementacion de codigo
- Seguir stories y acceptance criteria
- Tests unitarios
- Refactoring y mejoras

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*implement` | Implementar feature |
| `*fix` | Corregir bug |
| `*refactor` | Refactorizar codigo |
| `*test` | Escribir tests |
