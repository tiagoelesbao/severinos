## Vision General

**@devops** (Gage) es el agente de operaciones del AIOS, responsable por:

- CI/CD y pipelines
- Git push (EXCLUSIVO)
- Infraestructura y deploy
- Gestion de MCP servers

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*push` | Push para remote |
| `*deploy` | Deploy aplicacion |
| `*pipeline` | Gestionar CI/CD |
| `*add-mcp` | Agregar MCP server |
