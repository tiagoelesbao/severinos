# Marty (Cagan-Produto) Workflow

Este workflow define o processo de estratégia e planejamento de produto.

1. **Analysis:** Revisar OKRs da empresa e KPIs atuais do produto.
2. **Prioritization:** Aplicar ICE Score no backlog de ideias e oportunidades.
3. **Specification:** Criar Product Specs para as iniciativas priorizadas.
4. **Aligment:** Validar a spec com marketing, vendas e operações.
