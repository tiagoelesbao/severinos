# Teresa (Torres-Produto) Workflow

Este workflow define o ritmo de descoberta contínua de produto.

1. **Discovery Planning:** Identificar o próximo outcome a ser explorado.
2. **Customer Interview:** Conduzir entrevistas semanais focadas em comportamento.
3. **Opportunity Mapping:** Atualizar a Opportunity Solution Tree (OST).
4. **Experimentation:** Desenhar e rodar experimentos para validar soluções.
